# gitseek

Search GitHub projects directly from your terminal.

## Install

pip install gitseek

## Usage

Search projects:

gitseek ai tools

Example output:

[1] AutoGPT
⭐ 166k
An experimental open-source attempt to make GPT-4 fully autonomous.
https://github.com/Significant-Gravitas/AutoGPT

[2] Jarvis
⭐ 24k
LLM system connecting ML community tools.
https://github.com/microsoft/JARVIS

## Options

Limit results:

gitseek ai --limit 10

Open repository in browser:

gitseek ai --open 1

Clone repository:

gitseek ai --clone 2
